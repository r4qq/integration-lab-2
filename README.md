# integration-lab-2
### golang-blog

Celem laboratorium jest stworzenie lokalnej wersji aplikacji 'Blog' w golang oraz opanowanie pracy na gałęziach (Feature Branch Workflow) w systemie Git.

Przed uruchomieniem należy pobrać zależności:
```bash
go mod download
```

Uruchomienie projektu:
```bash
go run main.go
```
